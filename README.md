# Loophole AI Website

The official website for Loophole AI — a modern, responsive web experience showcasing our AI platform.

## Overview

This repository contains the source code for the Loophole AI marketing website, including:

- **Homepage** — Product overview and features
- **Models** — Available AI models and capabilities  
- **Pricing** — Subscription plans and options
- **Documentation** — User guides and API references
- **Blog** — News, updates, and insights
- **Support** — Help center and contact information

## Tech Stack

- HTML5
- CSS3 (custom styling)
- Vanilla JavaScript
- Responsive design (mobile-first)

## Project Structure

```
├── index.html          # Homepage
├── models.html         # Models page
├── pricing.html        # Pricing page
├── docs.html           # Documentation
├── blog.html           # Blog page
├── support.html        # Support page
├── styles.css          # Main stylesheet
├── script.js           # Main JavaScript
├── css/                # Additional stylesheets
└── assets/             # Images and media
```

## Development

This is a static website that can be served directly from any web server or CDN.
