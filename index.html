Plan to Execution.

Use any model in Plan or Execution mode — even open source models that don't natively support tool calling.

Plan Mode

Plan mode lets you map out your entire feature before writing a single line — define tasks, set context, and hand off to the agent with full confidence.

Execution Mode

Execution mode is a focused version of Agent mode — it follows your plan step by step, writing code, running tests, and committing changes autonomously.

How it works

Download & Open
Install in under 2 minutes. Built on Void, so your VS Code extensions, themes, and shortcuts work from day one — no setup, no migration.
Connect your AI
Paste your API key from OpenAI, Anthropic, or Gemini. Or skip it entirely and connect a local Ollama model — fully offline, no internet required.
Start building
Tab to complete, chat to refactor, or let the agent handle it. You write the vision — Loophole writes the code.
